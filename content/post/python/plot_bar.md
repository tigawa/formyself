- コード

    ```python
    pt2.plot(kind='bar', ylim=(0,1))
    ```

ylimは、ｙ軸の表示メモリの範囲を指定します。

今回は割合なので、0から1までをy軸のメモリとしています。

`pt2`は、[前回]({{< relref "pivot2.md" >}})の記事を参照してください。

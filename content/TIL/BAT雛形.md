## コード

```
setlocal
@echo off
CD /D "%~dp0"


endlocal
```

## 補足説明

@は、そこコマンド自体をECHOしないための設定
ECHO OFF だけだと ECHO OFFがOFFする前だからコンソールに出力される。
そのため＠ECHOにして防いでいる。

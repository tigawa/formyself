VSCodeの矩形選択のショートカットキーは次になります。


### MACのショートカット

```
［Shift］＋［Option］＋［Command］＋矢印キー
```

<!--more-->

### Windowsのショートカット

```
［Ctrl］＋［Shift］＋［Alt］＋矢印キー
```
